# Пет-проект GameShop
Это учебный проект интернет-магазина, созданный для практики работы с современным стеком React-разработки. Основной фокус был сделан на управлении сложным состоянием (Redux), типизации бизнес-логики и интеграции с внешним сервисом авторизации.

## 🎯 Цели проекта
Реализация полноценного Flow интернет-магазина (Каталог → Корзина → Оформление) с использованием следующих технологий:
- Vite + React + TS: Быстрая сборка и строгая типизация для предотвращения ошибок на этапе разработки.
- Redux Toolkit: Используется для управления состоянием корзины и данных заказа.
- Firebase: Выполняет роль бэкенда. Это позволило сосредоточиться на фронтенд-логике, имея при этом реальную базу данных товаров.
- LocalStorage API: Реализована синхронизация Redux-стора с локальным хранилищем, чтобы корзина не очищалась при обновлении страницы.

## 📈 Что я узнал/закрепил при разработки:
1) Настройка Firebase config и работа с SDK firestore.
2) Проектирование архитектуры Redux-слайсов.
3) Использование TypedUseSelectorHook и AppDispatch для полноценной поддержки TS в Redux.
5) Работа с асинхронными запросами (через createAsyncThunk или стандартные эффекты).

💻 Как запустить локально
- Клонируйте: `git clone https://github.com`
- Установите зависимости: npm install
- Создавать .env не нужно - внутри уже есть настроенный firebase (специально для примера работы).
